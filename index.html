Basic regex syntax

/ your regex goes here /

Regex are Case Sensitive

/ aeiOu zsHmr /

G flag at the end stands for global, meaning it searches the entire expression

/ regex /g

Regex Meaning
/[abc]/g Matches any of the includes characters
/[^abc]/g Matches any character NOT specified in brackets
/[^aeiou][aeiou]/g Matches a non-vowel (doesn't have to be a letter character) that is imediatelly followed by a vowel
/[A-Zf-s1-5]/g Matches any character withing specified that range
/./g Matches any character except linebreaks
/\w/g Matches any word character, equivalent to [A-Za-z0-9_]
/\w\w\w\w\w\w/g Would match 6 character words, including _
/\d/g Matches numeric digits, equivalent to [0-9]
/\s/g Matches any whitespace (spaces, tabs & linebreaks)
/+/g Matches one or more of preceding token
/*/g Matches 0 or more of the previous token
/a{5}/g Matches specific quantity of previous token, equivalent to /aaaaa/
/{1,3}/g Matches quantity within specific range. Between 1 and 3 in this case
/{4,}/g Matches the specified quantity of greater
/colou?r/g Matches 0 or 1 or previous token. Basically, makes some character optional
/\w+pp(y|ies)/g Matches all words with pp in it that are fallowed with either y or ies
/^/g Matches the beggining of a string
/$/g Matches the end of a string
/\+/g \ is an excape character. To escape and match characters with special meaning in regular expressions we must use \ (\+, \?, \^...)
/-?\d+\^-?\d+/g Matches an exponential expression, like: 4^2, 12^5, 5^-3, -15^6
/^-?\d+\^-?\d+$/g Checks that string ONLY contains a single exponential expression

Regex methods in JS

const someRegEx = /someregex/;

test() methods returns a boolean true or false

someRegEx.test('regextotest')

exec() method returns the information, the matches that if finds

someRegEx.exec('regextotest')
